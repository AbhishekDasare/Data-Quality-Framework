# Data Quality Framework

A PySpark-based data quality framework designed to validate incoming datasets before loading them into downstream analytics systems.

## Project Overview

This framework performs automated data quality checks including:

- Null value validation
- Duplicate record detection
- Schema validation
- Data type verification
- Quality reporting

## Tech Stack

- Python
- PySpark
- AWS S3
- AWS Glue
- Data Validation
- Data Engineering

## Key Features

- Automated validation pipeline
- Reusable quality checks
- Error reporting
- Scalable PySpark processing

## Business Benefits

- Improves data reliability
- Reduces reporting errors
- Prevents bad data from entering production systems
- Enhances downstream analytics accuracy
